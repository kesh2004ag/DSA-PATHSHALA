Linked List

A linked list is a linear data structure where elements (nodes) are stored in memory with each node pointing to the next node in the list.

Key Features of Linked List

  • Nodes: Each node contains data and a reference (or link) to the next node in the list.
  • Dynamic Size: Unlike arrays, the size of a linked list can grow or shrink during execution.
  • Non-contiguous Memory: Linked lists do not require contiguous memory allocation.

Example of Linked List in Python


class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

# Create Linked List
head = Node(10)
second = Node(20)
third = Node(30)

head.next = second
second.next = third

# Print Linked List
current = head
while current:
    print(current.data, end=" ")
    current = current.next  # Move to next node
