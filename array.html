Arrays

An array is a data structure that stores a collection of elements of the same data type in a contiguous memory location. Arrays are widely used to organize data in a structured way, making it easier to perform operations like accessing, updating, or iterating through the data.

What is an Array?

An array is a data structure that stores a collection of elements of the same data type in contiguous memory locations. Arrays are widely used to organize data in a structured way, making it easier to perform operations like accessing, updating, or iterating through the data.

Key Features of Arrays

  • Fixed Size: The size of an array is defined at the time of creation and cannot be changed.
  • Indexing: Elements in an array are accessed using indices. Indexing typically starts at 0 in most programming languages.
  • Homogeneous Elements: All elements in an array are of the same data type.
  • Contiguous Memory Allocation: All elements are stored in consecutive memory addresses.

Example of Stack in Python


    stack = []  # Using a list as an array

# Push operation
def push(value):
    stack.append(value)
    print(f"Pushed {value} onto the stack.")

# Pop operation
def pop():
    if len(stack) == 0:
        print("Stack Underflow! Stack is empty.")
        return None
    return stack.pop()

# Peek operation
def peek():
    if len(stack) == 0:
        print("Stack is empty.")
        return None
    return stack[-1]

# Display stack elements
def display():
    if len(stack) == 0:
        print("Stack is empty.")
    else:
        print("Stack:", stack)

# Example Usage
push(10)
push(20)
push(30)
display()
print("Popped:", pop())
print("Top element:", peek())
display()
